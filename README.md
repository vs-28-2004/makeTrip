# 🏠 Airbnb Clone

A full-stack Airbnb-inspired web application where users can browse, create, and manage property listings.

## 🚀 Features
- User Authentication
- Create, Edit & Delete Listings
- Property Search
- Image Uploads
- Responsive Design
- Booking System

## 🛠️ Tech Stack
- HTML, CSS, JavaScript
- Bootstrap
- Node.js & Express.js
- MongoDB
- EJS
- Cloudinary

## ⚙️ Installation

```bash
git clone https://github.com/your-username/airbnb-clone.git
cd airbnb-clone
npm install
npm start
